# Atte

## アプリケーションの目的と概要
人事評価を目的とした社員の勤怠を管理するためのシステム
![ホーム画面](ホーム画面.png)


##　アプリケーションURL


## 機能一覧
・会員登録、ログイン、ログアウト（Laravelの認証機能を利用）

・勤務開始、勤務終了（勤務終了時に日を跨いでいたら2日分の勤務として記録）

・休憩開始、休憩終了（1日に何度でも休憩可能）

・日付別勤怠情報取得、ページネーション（5件ずつ取得）


## 使用技術（実行環境）
